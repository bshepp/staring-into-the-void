# Roadmap

> Pre-promotion hardening checklist, ordered by leverage.  Items 1–5
> must be complete before executing the [outreach plan](OUTREACH.md);
> they are the minimum set that turns "interesting prototype" into
> "credible scientific tool" in the eyes of an LSST Science
> Collaboration reviewer.

## Out-of-scope (deliberately)

- GUI / web dashboard for individual source inspection — brokers do this
  better.
- General-purpose anomaly-detection framework — keep the scope tight to
  topological summaries on forced-photometry ensembles.
- Real-time per-source classification — this pipeline is *population*
  level; per-source ML is downstream of broker output.
